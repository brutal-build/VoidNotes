// features

Everything you need.
Nothing you don't.

No cloud. No subscriptions. No telemetry. Just pure, fast, local Markdown editing with the features that actually matter.

■

Local-first & Private

Every note saved as a plain .md file on your disk. No accounts, no servers, no sync. Your data belongs to you, always.

privacy by design
⇥

Markdown Native

Write in pure Markdown with syntax highlighting, live preview, split view, and full GFM support. Toggle between edit and preview instantly.

gfm / codeMirror 6
≡

Graph View & Links

Connect ideas with [[wiki links]], explore your knowledge graph visually, and discover backlinks automatically across all your notes.

second brain
⚡

Blazing Fast

Electron + React + TypeScript + Vite. Opens instantly, scrolls smoothly, auto-saves in 500ms. Under 100MB RAM. No bloat, ever.

lightweight

// two editions

Pick your Void.

Two flavors of the same philosophy. Choose the one that fits your workflow.

stable

Minimalistic

Clean, focused, distraction-free. The essentials - editor, preview, sidebar. Nothing more, nothing less.

  • Markdown editor + live preview
  • File explorer sidebar
  • 5 built-in themes
  • Auto-save, keyboard shortcuts
  • ~60MB RAM, instant launch
Download v1.0.0 main branch
v0.3.0 available

Expanded

Full power mode. Everything from Minimalistic plus advanced features for power users and knowledge workers.

  • Graph view & backlinks
  • Wiki links with autocomplete
  • Focus mode + daily notes
  • Tags, callouts, frontmatter
  • 6 themes (incl. macOS Glass)
  • Trash panel with preview
  • Vim keybindings, command palette
  • Code splitting, ~80MB RAM
